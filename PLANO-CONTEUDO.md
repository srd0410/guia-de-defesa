Este arquivo é o roteiro do acervo inicial. O Claude Code deve gerar os posts
seguindo o formato de `CONTEUDO.md`. Legenda:

- **[PILAR]** = artigo-pilar do cluster (guia grande; os satélites linkam de volta pra ele).
- **[AFILIADO]** = inserir `ProductCard` e marcar `affiliate: true`.
- **kw** = palavra-chave-alvo principal (use no título, na 1ª frase e nos H2 com naturalidade).
- Cada satélite deve conter **um link interno** para o artigo-pilar do seu cluster.

Regra de datas: todo post nasce com `pubDate`. Ao revisar/editar depois, adicione
`updatedDate` — o site mostra "atualizado em [data]" automaticamente (sinal de SEO).
